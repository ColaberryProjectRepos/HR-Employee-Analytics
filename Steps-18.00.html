Turnover % by Generation & Recruitment Source Mekko Chart 18

  • Click on custom objects and select vizlib library.  Drag and drop VizlibMekkoChart to the sheet and resize it. 
  • Click on click to add title and type "Turnover % by Generation & Recruitment Source"
  • Click on add dimension and select GenAgeRange. Unselect include null values.
  • Click on add dimension and select RecruitmentSource. Unselect include null values.
  • Click on add measure and select Turnover%. Sort it by Turnover%.
  • Go to presentation under appearance and show percentages. Select 10% for percentage formatting with a medium font size.
  • Select gray (#d4d4d4) for the border color. 
  • Go to colors and legend and color it by dimension. Use a color scheme then select strong20 and select reverse colors. Turn on show legend if its not on
