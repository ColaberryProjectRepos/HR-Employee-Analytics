Termination Reason Word Cloud Chart 17
  • Click on custom objects and go to Qlik Visualization Bundle. Drag Word Cloud Chart to the sheet from the library. Resize the object then change the title to ‘Termination Reason’.
  • Click on add dimension and select TermReason. Unselect include null values.
  • Click on add measure and select Employees.
  • Go to design under appearance. Set the orientation to 6; start angle should be -45 and end angle should be 45. Select reds for the scale color.
