Creating KPIs 11

Total Employees and Terminated Employees

  • Click on Charts and drag and drop KPI to the top left  of your sheet. Resize the KPI to be 2 blocks Vertically and 5 gridlines horizontally. 
  • Click on Add measure and select Employees. Unlink the measure and change the label to Total Employees.
  • Click on Second KPI and select Terminated.
  • Click on Appearance and go to color to change the colors. Turn off the conditional and library colors for the first measure then select Orchid or #a65392
  • Click on Second under measure and turn off the conditional and primary colors if they are on. Choose red or #f93f17

Turnover Rate


  • Click on Custom objects and select Vizlib Library.
  • Click and drag #1 Vizlib KPI total empployees KPI.
  • Click on Layer and then Add layer.  Type Chart under the layer name then select the line chart. 
  • Select [DateofTermination.autoCalendar.Year] for the dimension and Sum(Termd) for the measure. 
  • Sort by the Dimension numerically and in ascending order. Select area for the line styling. the line color should be black with an opacity of 22% while the area should be gray (#7b7a78) with a thickness of 3px and an area of 24. Select basis for the line type.
  • Add another layer and name it Turnover%. Select text and paste this where the text expression goes 
  • =((Count( {<EmploymentStatus ={'Terminated for Cause','Voluntarily Terminated'} >} EmploymentStatus ))  / (Count(EmploymentStatus)))
  • Format it as a number and select the percentage format of 100%. Choose black for the font color. 
  • Add another layer and call it Turnover Rate. Select text and type Turnover Rate where the text expression goes. The font color is black and the font size is 20. The horizontal align is center and the vertical align is top. 

Avg Months Worked


  • Click on Custom objects and select Vizlib Library.
  • Click and drag #1 Vizlib KPI under Turnover rate KPI.
  • Click on Layer and then Add layer.  Type Pie Chart under the layer name then select the pie chart. 
  • Select MaritalDesc for the dimension and Avg(12*(Year(DateofTermination)-Year(DateofHire))+(Month(DateofTermination)-Month(DateofHire)) ) for the measure. 
  • Sort by the Dimension Auto and the color mode should be by dimension. Show the label and select outside for the label position. The color should be black with a font size of 12.
  • Add another layer and name it AvgMonthsWorked. Select text and paste the Months worked measure  where the text expression goes 
  • =Avg(12*(Year(DateofTermination)-Year(DateofHire))+(Month(DateofTermination)-Month(DateofHire)) )
  • Format it as a number and select the first option that shows 10,000. Choose black for the font color. 
  • Add another layer and call it Title. Select text and type Avg Months Worked where the text expression goes. The font color is black and the font size is 20. The horizontal align is center and the vertical align is top. 

Detailed Sheet button


  • Go to charts then drag and drop button below Avg Months worked KPI. 
  • Go to properties and select go to next sheet for the navigation.
  • Go to appearance and select general. Type Detailed Sheet under label.
  • Select bold and align center under Font styling.
  • Select Orchid or #8e477d for the  background color.
  • Turn the icon on and type forward icon type. Position it on the left side.

Net Promoter Score


  • Click on Custom objects and select Vizlib Library.
  • Click and drag #1 Vizlib KPI under the detailed sheet button.
  • Click on Layer and then Add layer.  Type Icon under the layer name then select the icon visualization. 
  • Type fa-star for icon type and black for the font color. The horizontal and vertical icon align should be in the center.
  • Position the icon to the center.
  • Add another layer and name it NPS. Select text and paste the NPS measure  where the text expression goes 
  • Format it as a number and select the first option that shows 10,000. Choose black for the font color then align it vertically to the top and horizontally in the middle. 
  • Add another layer and call it Title. Select text and type Net Promoter Score where the text expression goes. The font color is black and the font size is 20. The horizontal align is center and the vertical align is top. 

Male And Female Percentages


  • Click on Custom objects and select Vizlib Library.
  • Click and drag #1 Vizlib KPI under the detailed sheet button.
  • Click on Layer and then Add layer.  Type Title Male under the layer name then select the text visualization.
  • Type Male where the text expression goes. The horizontal alignment should be center while the vertical should be top aligned.
  • Select blue or #142b88 for the font color and drag the layer to the top left of the KPI
  • Add your second layer and name it Title Female  then select the text visualization.
  • Type Female where the text expression goes. The horizontal alignment should be center while the vertical should be top aligned.
  • Select Red or #931c31 for the font color and drag it to the top right of your KPI.
  • Add the 3rd layer and name it Male calc. Select text and paste the Male measure where the text expression goes. 
  • Format it as number and num format should be 100%. 
  • The horizontal alignment should be center while the vertical should be top aligned.
  • Select blue or #142b88 for the font color and drag the layer and place it under Male title. 
  • Add the 4th layer and name it Female calc. Select text and paste the Female measure where the text expression goes. 
  • Format it as number and num format should be 100%. 
  • The horizontal alignment should be center while the vertical should be top aligned.
  • Select blue or #931c31 for the font color and drag the layer and place it under Female title.
  • Add the last layer and name it Icon. Type fa-user for icon type and paste  "If(sum(Gender='Female')< Sum(Gender='Male'), Red(),Blue())" where the color expression should go. 
  • The horizontal alignment should be center while the vertical should be top aligned.

Main Sheet button( for the second sheet)


Create a new sheet and name it Details. Copy all of the KPIs you created in the first sheet and paste it to the second sheet. Deleted the detailed sheet button and replace it with Main sheet button. The steps to create the button are as follows:

  • Go to charts then drag and drop button below Avg Months worked KPI. 
  • Go to properties and select go to previous sheet for the navigation.
  • Go to appearance and select general. Type Main Sheet under label.
  • Select bold and align center under Font styling.
  • Select Orchid or #8e477d for the  background color.
  • Turn the icon on and type Back icon type. Position it on the left side.
