Creating Applications and Transform The Data 8

You are about to create a dashboard for Human Resource Employee Analytics and this is what it should look like.

The following steps will instruct you on how to recreate the dashboard from start to finish.

We will start by opening Qlik Sense.

Creating Applications

1.         Click "Create New App"

2.         Name the application, we named it "Human Resource Analytics"

3.         Click "Create" and then click "Open App"

4.         Download the attached Zip file to use as a starting point and save the file to your computer.

5.         Click on  Script Editor then click on Create connection. Type Human Resource Analytics at the bottom of the popup screen and click on create.  

6.       Click on Select Data under your new connection and select the file you downloaded. Click on Insert Script and load the script. 

Now that we have loaded the data into Qlik Sense App, we need to transform it to better suit our needs.


      Transform the data by using  The code below 


1.   Replace GenderID with     IF( GenderID = 1,'Male','Female') As Gender,

2.    Create Age and AgeRange after the DOB column      FABS(Age(Today(),DOB)) As Age,

    Dual(Replace(Class(Fabs(Age(today(), DOB)),5),'<= x <','-'),Class(Fabs(Age(today(), DOB)),5)) As  AgeRange,

3.    Load the data in the script editor and then load it in the Data Manager.
4.    After loading the data click on Edit Sheet.  Click on  My new sheet and click on the detail icon that is on the bottom right of the sheet that was created. Click on the edit icon to change the name of the sheet to Human Resource Analytics then save it.
