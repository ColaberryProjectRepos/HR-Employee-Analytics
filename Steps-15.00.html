Turnover by Dept 15

  • Go to Custom objects and drag Combo chart to the sheet. Resize the object then change the title to ‘Turnover by Year’.
  • Click add dimension and select DeptPositionEmp. Unselect include null values. Click on Add measure and Select Active%.
  • Change the formatting to number formatting and select the percentage with no decimal values.
  • Make the color gray or #7b7a78 and make the opacity 75%.
  • Add  a second measure and select Turnover%. 
  • Change the formatting to number formatting and select the percentage with no decimal values.
  • Select Bar for the representation. The bar color should be black. Go to presentation under general and overlap for the bar type. 
  • Add a new measure and select NPS. The axis position is secondary and select symbol for representation. Pick star as a representation and make the color ='#b73d2d'.
  • Change the formatting to number formatting and select the number with no decimal values.
  • Add the last measure and select Months worked. 
  • Change the formatting to number formatting and select the number with no decimal values.
  • The axis position is secondary and select line for representation. The line style should be solid and the curve style should be linear.
  • Show the data point and the type should be area. The color for the area should be #914980.
  • Go to sorting and sort it by Turnover% followed by Active%.
  • Click on Y-axis (left) under appearance and choose black for the font and line color.
  • Then click on Y-axis(right).  Select purple (#914980)  for the font and axis line color. The font size should be 12.
