Location Map 13

Location

You are about to create a map that looks like this.

  • Go to charts and drag Map to the sheet. Resize the Map and rename it Location.
  • Click on Add Layer on the properties window and select point layer.
  • Type "Location" where it says click to add title on the map.
  • Go to the properties window and click on add to add a dimension. Type Zip and select it under fields.
  • Click on location then select zip under field location.
  • Unselect auto under scope for location. Select postal code/zip code under location type then type 'USA' under country.
  • Click on size and shape then size it by employees. Select bubble under shape.
  • Go to colors and turn off the custom colors and select by measure. 
  • Measure it by employees then turn on the master measure format and library colors.
  • Go to options and select general. select auto for show legend. Show layer under layer display.
