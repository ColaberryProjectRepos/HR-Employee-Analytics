Gender Performance Score by Department 16


  • Click on custom objects and go to Vizlib library. Drag Sankey chart to the sheet from the library. Resize the object then change the title to ‘Gender Performance Score by Department’.
  • Click on add dimension and select DeptPosition. Unselect include null values.
  • Click on add dimension and select PerformanceScore. Unselect include null values.
  • Select Count(Gender) to be your measure. In order to make the label dynamic we have to use an expression that will change depending on which gender is selected. If no gender is selected then it will show employees.
  • Paste this expression to the label. =If(Gender = 'Male','Male', If(Gender = 'Female','Female','Employees'))
  • Go to colors under appearance and color it by node. Select classes for coloring type and use a color scheme.
  • Select Strong20 from the color scheme and select reverse colors. 
  • Go to dimension titles under appearance and hide it.
